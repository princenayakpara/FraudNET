## üí° Innovation & Creativity
We moved beyond simple `if-else` scripts.
- **Hybrid AI Engine**: Combines rule-based heuristics (for immediate threats) with an **Unsupervised Learning Model (Isolation Forest)** to detect subtle system anomalies that static rules miss.

- **Glassmorphism UI**: A modern, fluid interface that enables complex data visualization without overwhelming the user.

## üõ†Ô∏è Tech Stack Strategy (The "Hardware/Software Selection")
We chose a **Lightweight & Powerful** stack to ensure the manager itself doesn't become bloatware.

| Component | Choice | Justification |
| :--- | :--- | :--- |
| **Backend** | **Python (FastAPI)** | High performance, async capabilities, and native support for AI/ML libraries (`scikit-learn`, `pandas`). |
| **AI Model** | **Isolation Forest** | Perfect for anomaly detection in time-series data (CPU/RAM usage) without needing massive labeled datasets. |
| **Frontend** | **Vanilla JS + CSS3** | Zero-dependency approach ensures 100% control over rendering performance and ultra-low memory footprint. |
| **System** | **Windows API (winreg)** | Direct Deep-Level access to system registry for true optimization, not just cosmetic changes. |
